Браузерние інтерфейси:

  • DOM - Document Object Model обектна медель документа, предназначен для изменения страниц существуют такие обекти - документ (содержимое страници), ивент - собития на странице,
  • BOM - Browser Object Model - обектная модель браузера, тоесть набором дополнительних обектов, которие реализуют различние интерфейси,и позводляют работать совсем, кроме документа
